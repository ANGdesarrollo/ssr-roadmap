# Pista 3 — Duplicados y robustez

Para eliminar duplicados en `files`: `[...new Set(files)]` en TS/JS.

Para ignorar líneas malformadas: si la primera línea de un bloque no tiene el formato esperado (ej: no tiene espacio), simplemente no la incluyas en el resultado. Un `try/catch` o un `if` simple sirven.

Para el caso de bloque vacío o con solo espacios: filtrá los bloques antes de procesarlos con `.filter(block => block.trim() !== '')`.
