# Pista 2 — Parsear un bloque

Cada bloque tiene esta forma:
```
a1b2c3d feat: add order validation
src/orders/order.service.ts
src/orders/order.dto.ts
```

La primera línea siempre es `hash mensaje`. Las siguientes son archivos (0 o más).

Tip: `split('\n')` en el bloque te da las líneas. La primera es especial, el resto son archivos.

Para el hash: los primeros 7 caracteres. Para el mensaje: todo lo que viene después del primer espacio.
